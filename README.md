# stringheist
String Playful

## Setup

### Create and enter project directory
```bash
mkdir stringheist
cd stringheist
```

### Create virtual environment
```bash
python -m venv .venv
```

### Activate venv
**macOS/Linux:**
```bash
source .venv/bin/activate
```

**Windows PowerShell:**
```powershell
.venv\Scripts\Activate.ps1
```
